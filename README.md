# 🤖 Rule-Based AI Chatbot

A conversational AI chatbot built using Python with rule-based pattern matching techniques.
Built as **Task 1** of the **CodSoft Artificial Intelligence Internship**.

---


## 📌 Features

- 💬 Natural conversation flow
- 🔍 Pattern matching using Regular Expressions (Regex)
- 🕐 Real-time date and time responses
- 🤖 Answers questions about AI, ML, and Python
- 😄 Tells jokes
- 🔄 Continuous conversation loop
- ❌ Clean exit with 'quit' or 'bye'

---

## 🛠️ Tech Stack

- **Language** — Python 3.11
- **Library** — `re` (Regular Expressions)
- **Library** — `datetime`

## 🧠 How It Works

```
User types input
      ↓
Input converted to lowercase
      ↓
Regex pattern matching checks conditions
      ↓
First matching condition returns response
      ↓
Bot prints the response
      ↓
Loop continues until user types 'quit'
```
